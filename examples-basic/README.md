# exam-zh 入门示例

本目录包含 exam-zh 的入门示例，专为新手设计，循序渐进地展示核心功能。

## 📝 示例列表

### 00-minimal.tex - 最小示例（约10行）
- **目标**：最简单的可编译文档
- **内容**：标题 + 1道题目
- **学习要点**：基本文档结构

### 01-first-exam.tex - 第一份试卷（约30-40行）
- **目标**：创建第一份完整试卷
- **内容**：标题、科目 + 选择题2道 + 填空题1道 + 解答题1道
- **学习要点**：基础题型使用

### 02-math-basic.tex - 基础数学试卷（约80-100行）
- **目标**：完整的数学试卷结构
- **内容**：选择题10道、填空题5道、解答题3道、草稿纸
- **学习要点**：数学试卷排版、答案控制

### 03-chinese-basic.tex - 基础语文试卷
- **目标**：语文学科特性展示
- **内容**：现代文阅读、古诗文、作文
- **学习要点**：材料文章、古诗排版

### 04-english-basic.tex - 基础英语试卷
- **目标**：英语学科特性展示
- **内容**：听力、阅读理解、写作
- **学习要点**：作文框、选项排版

## 🚀 使用方法

### 方式一：命令行编译
```bash
cd examples-basic
xelatex 00-minimal.tex
```

### 方式二：使用 IDE
在 VS Code、TeXstudio 等编辑器中打开 `.tex` 文件，选择 XeLaTeX 编译即可。

### 方式三：在线编译
可以上传到 Overleaf 或 TeXPage 等在线平台编译（记得在文档开头添加 `\let\stop\empty`）。

## 💡 学习建议

1. **按顺序学习**：从 00 到 04 逐个尝试
2. **修改练习**：在示例基础上修改内容，加深理解
3. **对比效果**：修改参数，观察输出变化
4. **复制使用**：找到合适的示例，直接复制作为模板
